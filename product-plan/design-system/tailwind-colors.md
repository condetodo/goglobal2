# Tailwind Color Configuration

## Color Choices

- **Primary:** `blue-900` — Used for buttons, links, key accents
- **Secondary:** `red-700` — Used for tags, highlights, secondary elements
- **Neutral:** `stone` — Used for backgrounds, text, borders

## Usage Examples

Primary button: `bg-blue-900 hover:bg-blue-800 text-white`
Secondary badge: `bg-red-100 text-red-800`
Neutral text: `text-stone-600 dark:text-stone-400`
